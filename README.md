

how to install the project  

```
git clone https://github.com/chaman56/aikart.git
```  

## initialize frontend
``` 
cd aikart/frontend 
```  
``` 
npm install 
```  


## initialize backend
``` 
cd ../backend 
```  
``` 
npm install 
```  

## run frontend  
``` 
npm run dev 
```  


## run backend 
``` 
npm start 
```
